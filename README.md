# Automatic-xhs
浏览器插件-红薯养号助手，小红薯笔记操作自动化！
功能：自定义关键词，自定义运行时长，点赞，评论，收藏！
免费使用，防止盗卖，不开源！
随缘更新！

# 插件使用方法：
第一种，直接拖动crx文件到浏览器。
第二种，浏览器打开chrome://extensions/  把开发者模式打开，加载已解压的扩展程序，选择项目文件夹。
